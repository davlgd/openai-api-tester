# OpenAI API Tester

OpenAI API Tester is a tool designed to interact with APIs compatible with OpenAI's format.

## Installation

To install the necessary dependencies, use the `uv` package manager:

```bash
uv install
```

## Development

To run the application in development mode, use the following command:

```bash
uv run uvicorn main:app --reload
```

## Production

To run the application in production mode, use the following command:

```bash
uv run uvicorn main:app --host 0.0.0.0 --port 9000
```

## Usage

This project aims to provide a seamless interface for interacting with various APIs that are compatible with OpenAI's API.
